## Project
# StockSense – Inventory Management System

## 📌 Project Overview
StockSense is a responsive Inventory Management System designed to help users manage products, monitor stock levels, and organize inventory records efficiently.

---

## 🎯 Objectives
- Manage inventory efficiently  
- Add and view products  
- Track stock availability  
- Demonstrate modern web design principles  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## ✨ Features
- Dashboard interface  
- Product listing page  
- Add product form  
- Responsive sidebar navigation  
- Search and filtering system
